USB Video reformatter

Converts video files to a format used by most (all?) TVs.

Files of type webm, avi, mkv and mp4 are converted to mp4 (brand mp41), internal video stream is converted to AVC and internal audio stream is converted to AAC

This code uses ffmpeg and mediainfo - see installation for details.

The code loops through all nested folders and then sends each videofile to mediainfo. This takes a short while to analyse the file's coding and report back. However ffmpeg will then take considerably longer to process each file.

When completed, the time taken for each file to be processed will be displayed.

THIS IS IRREVERSIBLE! If you want to keep the old files, check you have enough disk space for the new files
